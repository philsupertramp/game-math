\doxysubsection{Detailed Description}
Decision (Bi-\/)Tree implementation
\begin{DoxyItemize}
\item The constructor allows providing a decision\+\_\+method for the impurity measure that is used when selecting a good split
\item The {\itshape information\+\_\+gain(parent, left\+\_\+child, right\+\_\+child) calculates the information gain value of a split between a parent and two children}
\item {\itshape The \+\_\+best\+\_\+split(\+X, y) function calculates the best splitting parameters for input features X and a target variable y
\begin{DoxyItemize}
\item It does so by iterating over every column in X and every thresold value in every column to find the optimal split using information gain
\end{DoxyItemize}}
\item {\itshape The \+\_\+build(\+X, y, depth) function recursively builds a decision tree until the data is \+\_\+non splittable}
\item The fit(\+X, y) function calls the \+\_\+build() function and stores the built tree to the constructor
\item The \+\_\+predict(x) function traverses the tree to classify a single instance
\item The predict(\+X) function applies the \mbox{\hyperlink{classDecisionTree_ad6d9caca64c5ff9d73e710e86b52ed46}{\+\_\+predict()}} function to every instance in matrix X 
\end{DoxyItemize}
