\doxysubsection{Detailed Description}
Numerical differentiation methods to approximate the first order derivative using support values.

\$\$f\textquotesingle{}(x) = ?\$\$

Implemented quotients\+:
\begin{DoxyItemize}
\item Forward \$\$ \textbackslash{}frac\{f(x+h)-\/f(x)\}\{h\} \$\$
\item Backward \$\$ \textbackslash{}frac\{f(x)-\/f(x-\/h)\}\{h\} \$\$
\item Center \$\$ \textbackslash{}frac\{f(x+h)-\/f(x-\/h)\}\{2h\} \$\$
\end{DoxyItemize}
