\doxysubsection{Detailed Description}
Binary Decision Tree Node.

Avaibale in two types\+:
\begin{DoxyItemize}
\item LEAF\+: Leaf node holding final class label
\item DECISION\+: Node holding two children that split a subset of the train data 
\end{DoxyItemize}
