\doxysubsection{Detailed Description}
Represents \mbox{\hyperlink{classPerceptron}{Perceptron}} definition by Frank Rosenblatt

\href{https://en.wikipedia.org/wiki/Perceptron}{\texttt{ Wikipedia}}

Essentially it can be used to build binary classification One might for example want to replace some very complex binary operation which is a combination of several broad operations like OR$\vert$\+AND$\vert$\+XOR$\vert$\+NOT ...

In order to use the perceptron classifier one needs access to the full set of possible transformations.

THIS IS NOT A PREDICTOR

Mainly implemented for academic use
