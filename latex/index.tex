\hypertarget{index_autotoc_md25}{}\doxysubsection{Extensions}\label{index_autotoc_md25}
Currently there are two different implementations of matrices/vectors \hypertarget{index_autotoc_md26}{}\doxysubsubsection{Implementation 1)}\label{index_autotoc_md26}
Game dev oriented. 2, 3 and 4 dimensional vector/matrix implementations. ~\newline
 Tightly packed data to allow minimal memory usage. see {\ttfamily ./include/math/vec} and {\ttfamily ./include/math/mat} for implementation details.\hypertarget{index_autotoc_md27}{}\doxysubsubsection{Implementation 2)}\label{index_autotoc_md27}
{\ttfamily \mbox{\hyperlink{classMatrix}{Matrix}}$<$T$>$} (see {\ttfamily ./include/math/\+Matrix.h}) represents a {\ttfamily m x n}-\/\mbox{\hyperlink{classMatrix}{Matrix}} of data of type {\ttfamily T}

a specific data science and numerics implementation. ~\newline
 Capable of Matrix-\/\+Vector and Matrix-\/\+Multiplication, it also contains data science specific algorithms and statistical methods. \hypertarget{index_autotoc_md28}{}\doxysubsubsection{Available extensions\+:}\label{index_autotoc_md28}
