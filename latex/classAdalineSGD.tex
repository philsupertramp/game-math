\doxysubsection{Detailed Description}
Adaline linear neuron implementation using statistic gradient decent for training of a vector of weights and a single bias.

weights\+: shape \mbox{[}N+1 x 1\mbox{]} bias\+: shape \mbox{[}1, 1\mbox{]}, first element of weights

Note\+: only allows binary classification and uses a single layer of weights see the description here \mbox{[}1\mbox{]} I found it very easy to understand and implement.
